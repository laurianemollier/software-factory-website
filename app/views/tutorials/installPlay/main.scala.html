Installer Play Framework

L'installation de sbt se fait en deux temps

  • Installer Java (du moins vérifier que la JDK 1.8 est bien installée)
  • Installer Sbt

Alors c'est partie!

Installer Java

Pour fonctionner, Play framework a besoin de Java 1.8.

Pour vérifier que nous avons bien la bonne de JDK, ouvrons un terminal et tapons en ligne de commande:

@widgets.course.code(bash) { java -version }

Si quelque chose comme ça s’affiche, c’est que tout est bon!

@widgets.course.code(bash) { java version "1.8.0_121" Java(TM) SE Runtime Environment (build 1.8.0_121-b13) Java HotSpot(TM) 64-Bit Server VM (build 25.121-b13, mixed mode) }

Sinon, c’est que tu n’as pas de JDK, ou alors pas la bonne version. Il faut donc aller l’installer.
Toutes les infos se trouvent sur le site internet d’Oracle
